# New-AMProcessCondition

## SYNOPSIS
Creates a new AutoMate Enterprise process condition.

## SYNTAX

```
New-AMProcessCondition [-Name] <String> -ProcessName <String> [-Action <AMProcessTriggerState>] [-TriggerOnce]
 [-Started] [-Wait] [-Timeout <Int32>] [-TimeoutUnit <AMTimeMeasure>] [-TriggerAfter <Int32>] [-Notes <String>]
 [-Folder <Object>] [-Connection <Object>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
New-AMProcessCondition creates a new process condition.
