# New-AMUser

## SYNOPSIS
Creates a new AutoMate Enterprise user.

## SYNTAX

```
New-AMUser [-Name] <String> [-Notes <String>] [-Folder <Object>] [-Connection <Object>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
New-AMUser creates a new user object.

## EXAMPLES

### EXAMPLE 1
```
# Create new user
```

New-AMUser -Name David
