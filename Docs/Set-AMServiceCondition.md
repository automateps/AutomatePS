# Set-AMServiceCondition

## SYNOPSIS
Sets properties of an AutoMate Enterprise service condition.

## SYNTAX

```
Set-AMServiceCondition [-InputObject] <Object> [[-ServiceName] <String>] [[-Action] <AMServiceTriggerState>]
 [-Wait] [[-Timeout] <Int32>] [[-TimeoutUnit] <AMTimeMeasure>] [[-TriggerAfter] <Int32>] [[-Notes] <String>]
 [[-CompletionState] <AMCompletionState>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Set-AMServiceCondition modifies an existing service condition.
