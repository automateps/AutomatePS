# Set-AMProcessCondition

## SYNOPSIS
Sets properties of an AutoMate Enterprise process condition.

## SYNTAX

```
Set-AMProcessCondition [-InputObject] <Object> [[-ProcessName] <String>] [[-Action] <AMProcessTriggerState>]
 [-TriggerOnce] [-Started] [-Wait] [[-Timeout] <Int32>] [[-TimeoutUnit] <AMTimeMeasure>]
 [[-TriggerAfter] <Int32>] [[-Notes] <String>] [[-CompletionState] <AMCompletionState>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Set-AMProcessCondition modifies an existing process condition.
