# Set-AMDatabaseCondition

## SYNOPSIS
Sets properties of an Automate database condition.

## SYNTAX

```
Set-AMDatabaseCondition [-InputObject] <Object> [[-DatabaseType] <AMDatabaseTriggerType>] [[-Server] <String>]
 [[-NotificationPort] <Int32>] [[-Database] <String>] [[-Table] <String>] [[-UserName] <String>] [-Insert]
 [-Delete] [-Update] [-Drop] [-Alter] [-Wait] [[-Timeout] <Int32>] [[-TimeoutUnit] <AMTimeMeasure>]
 [[-TriggerAfter] <Int32>] [[-Notes] <String>] [[-CompletionState] <AMCompletionState>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Set-AMDatabaseCondition modifies an existing database condition.
