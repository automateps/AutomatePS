# New-AMLogonCondition

## SYNOPSIS
Creates a new Automate logon condition.

## SYNTAX

```
New-AMLogonCondition [-Name] <String> [-User <String[]>] [-Wait] [-Timeout <Int32>]
 [-TimeoutUnit <AMTimeMeasure>] [-Notes <String>] [-Folder <Object>] [-Connection <Object>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
New-AMLogonCondition creates a new idle condition.
